# 症例画像の追加フォルダ

## 使い方（超かんたん）
1. Instagram / スマホから画像を保存
2. このフォルダ（images/incoming/）にドラッグ＆ドロップ
3. Claudeに「眉の症例◯枚追加した」と伝える（キャプションあれば貼る）

## カテゴリ（data-category で振り分け）
eyebrow / eyeliner / lip / hairline / scalp / amarapink / paramedical

## カテゴリを分けたいとき（任意）
サブフォルダを作って入れてもOK:
  images/incoming/eyebrow/  → 眉
  images/incoming/lip/      → リップ
など

Claudeがリネーム→images/へ移動→cases.htmlへ挿入→commitまで自動で行います。
処理後、このフォルダは空になります。
